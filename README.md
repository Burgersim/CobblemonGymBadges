# Cobblemon Gym Badges

## Description

This mod adds a Badge Press for crafting Gym Badges. Only players with the Gym Leader Role can craft the corresponding badge(s).

## Features

- Badge Press Block
  - only lets Players with Gym Leader Roles craft the corresponding Badges
- Gym Leader Roles
- Datapacks / Resourcepacks for adding custom badges
- JEI/EMI-Integration
  - only craftable recipes should be visible to each Player
